## 🎯 核心功能

### MD5 计算功能
```javascript
computeMD5() - 计算 MD5 哈希值
- 支持小写/大写输出
- 支持 32位/16位输出
- 输入验证
- 错误处理
```

### 辅助功能
- `clearMD5()` - 清空所有输入和输出
- `copyMD5()` - 复制结果到剪贴板

## 🚀 使用方法

### 基本使用
1. 打开 `md5.html`
2. 输入要转换的文本
3. 选择大小写格式（小写/大写）
4. 选择输出长度（32位/16位）
5. 点击"计算"按钮
6. 点击"复制"复制结果

### 选项说明
- **小写**: 输出小写 MD5 哈希值
- **大写**: 输出大写 MD5 哈希值
- **32位**: 输出完整的 32 位 MD5 哈希值
- **16位**: 输出中间 16 位的 MD5 哈希值
